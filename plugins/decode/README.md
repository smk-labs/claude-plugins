# decode

A senior-style narrated tour of your codebase, in product language. Persian-first.

قصهٔ کدبیس به زبان محصول، با روایت یه سینیور: برای مدیر محصولی که کد نمی‌خونه ولی می‌خواد نقشهٔ کامل توی ذهنش شکل بگیره و کم‌کم دولوپر بشه.

## سه حالت

- `/decode src/checkout.ts`: قصهٔ کوتاه همون فایل توی چت: چه‌کاره‌ست، به کی وصله، و اگه قلب سیستمه، زیر کاپوتش چی می‌گذره.
- `/decode atlas` یا `/decode src`: تور HTML مستقل: قصهٔ دو دقیقه‌ای، درخت فایل مثل VS Code با وزن‌دهی (قلب، مهم، سیم‌کشی، جانبی)، کانواس خلاصه برای هر فایل با کلیک، و درس‌های کوچیک مهندسی. خروجی: `atlas.html` در ریشهٔ ریپو.
- `/decode diff` یا `/decode PR 42`: ترجمهٔ تغییرات به زبان محصول، مثل ریلیز نوت، با فهرست «چی رو تست کنیم».

## قانون

قصه، نه دانشنامه: به کد نابرابر فضای برابر نده. فایل سیم‌کشی یک جمله می‌گیره («فقط پیامک می‌فرسته. همین.») و فایل قلب پاراگراف و ارتباط‌ها و زیر کاپوت با لنگر `file:line`. کلمات معماری آزاد، کلمات سینتکس ممنوع، روایت خط‌به‌خط ممنوع.
